FAQ

Common questions about YourMove

How does YourMove work?

YourMove uses VR sensors to track body movements in real-time. Our AI analyzes this data to detect stress patterns, tremors, and focus levels, then adapts therapy sessions accordingly for optimal outcomes.

Is it safe for children?

Absolutely. YourMove prioritizes child safety. All sessions are monitored by licensed therapists, and the system automatically intervenes if excessive stress is detected.

What age range is appropriate?

YourMove is optimized for children aged 4-12 with autism spectrum disorder. The system adapts to different developmental levels and diagnosis severity (Level 1, 2, or 3).

How do doctors monitor sessions?

Therapists use our secure dashboard to view real-time body stress maps, live charts, and AI-generated insights during each session through an intuitive web interface.
