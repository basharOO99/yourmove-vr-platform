Clinical Features

Built for practising clinicians

Every feature is designed to support, not replace, clinical judgement.

Patient Stability Index
0–100 composite score combining focus, arousal, and tremor data. Updated every frame. Quantifies session-level therapeutic state in a single interpretable metric.
Z-Score Anomaly Detection
Per-sensor rolling baselines computed via Welford's online algorithm. Standardised z-scores flag statistically significant deviations at 2σ, 2.5σ, 3σ, and 4σ thresholds.
Predictive Stress Modelling
OLS linear regression and EWMA ensemble model forecasts stress levels at 30, 60, and 120-second horizons. Enables pre-emptive intervention before outburst events.
AI Confidence Scoring
Every AI decision is accompanied by a confidence score based on buffer fill level, signal-to-noise ratio, and model R². Clinicians are always informed of analytical reliability.
13-Sensor Body Heatmap
Real-time 5-state colour map (Normal → Critical) for all 13 body-part sensors. EMA-smoothed values prevent visual noise. Hovering reveals z-score and trend slope.
Session Report Export
One-click PDF export of the live dashboard including all charts, stability score, risk classification, anomaly log, and auto-generated session summary for clinical records.

Research Ready

Designed for clinical research and audit

YourMove produces structured, exportable data that supports academic research into VR-based behavioural therapy for ASD.

Data Architecture
Structured session logging
Every session is logged to indexed SQLite tables with millisecond timestamps, AI decisions, confidence scores, and raw sensor aggregates. Export via REST API in JSON format for downstream analysis.
GET /api/session/export?session_id=…
GET /api/session/history?patient_id=…
Audit Trail
Structured AI decision logging
Every AI command, anomaly event, and risk classification is logged as structured JSON to a dedicated audit logger. Designed for GDPR-compliant clinical documentation in accordance with medical device traceability requirements.
{"event":"ai_command","ts":…,
 "command":"slow_down",
 "severity":"medium",
 "anomalies":2,"confidence":0.82}
